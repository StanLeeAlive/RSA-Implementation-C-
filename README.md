## Цель работы

Программно реализовать криптосистему RSA и экспериментально убедиться в корректности и работоспособности её основных алгоритмов при длине модуля 4096 бит.

---

## Реализованные алгоритмы

### Основные алгоритмы 

| Алгоритм | Описание                           | 
| -------- | ---------------------------------- |
| Gen  | Генерация ключей (p, q, n, φ(n), e, d) |
| Encr | Шифрование: `c = m^e mod n`            | 
| Decr | Расшифрование: `m = c^d mod n`         | 

### Вспомогательные алгоритмы

| Алгоритм     | Описание                                   | 
| ------------ | ------------------------------------------ | 
| ExEu     | Расширенный алгоритм Евклида                   | 
| ModPow   | Возведение в степень по модулю (справа налево) |
| GenPrime | Генерация простых чисел (вариант А)            | 
| IsPrime  | Тест Миллера-Рабина (100 раундов)              |

### Оптимизации (по желанию)

 Функция Кармайкла вместо функции Эйлера 
 Сохранение p и q для ускорения расшифрования (китайская теорема об остатках) 

### Дополнительное задание

  Схема дополнения OAEP

## Сборка и запуск

### Требования

 .NET 9.0 SDK или новее
 Поддержка `System.Numerics.BigInteger`

### Сборка проекта

```bash
cd CryptoLAB_RSA
dotnet restore
dotnet build
```

### Запуск программы

```bash
dotnet run
dotnet run --configuration Release
```

##  Описание автотестов

| Тест                  | Проверяемый функционал         | Критерий успеха                 |
| --------------------- | ------------------------------ | ------------------------------- |
| `TestExtendedGCD`     | Расширенный алгоритм Евклида   | `a·x + b·y == gcd(a,b)`         |
| `TestModPow`          | Возведение в степень по модулю | Совпадает с `BigInteger.ModPow` |
| `TestPrimality`       | Миллер-Рабин                   | Корректная классификация        |
| `TestKeyGeneration`   | Генерация ключей RSA           | `e·d ≡ 1 (mod φ(n))`            |
| `TestEncryptionCycle` | Полный цикл                    | `Decr(Encr(m)) == m`            |
